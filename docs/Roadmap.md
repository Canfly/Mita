# Roadmap

## P0 - Концепт
Критерии перехода:
- Канон и глоссарий утверждены.
- Определены базовые спецификации.

## P1 - Прототип документов
Критерии перехода:
- Опубликованы Interaction Ledger и Marketplace Event Spec.
- Определены Tier правила.

## P2 - Лабораторные пилоты
Критерии перехода:
- Пилот Campus Mesh описан.
- Определены метрики и prereg шаблоны.

## P3 - Ограниченный релиз
Критерии перехода:
- Верифицированы базовые метрики устойчивости.
- Прошел внешний аудит threat model.

## P4 - Расширение доменов
Критерии перехода:
- Запущены несколько Guardian доменов.
- Налажен арбитраж и апелляции.

## P5 - Масштабирование
Критерии перехода:
- Проверена устойчивость к блокировкам.
- Поддержка нескольких рынков и регионов.
