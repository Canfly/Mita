# Node Lexicon

## Основные роли
- Nest Node - базовый узел на обычном компьютере.
- Relay Node - транспортный усилитель - NAT traversal, rate limiting.
- Index Node - поиск и индексация по разрешенным метаданным.
- Vault Node - хранение артефактов сделки - шифрование и TTL.
- Guardian Node - домен политики - Tier правила и санкции.
- Arbiter Node - арбитраж - споры, апелляции, вердикты.
- Oracle Node - подтверждения физического мира.
- Specter Node - аналитический сенсор - сигналы антифрода.
