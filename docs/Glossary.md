# Glossary

| Термин | Определение |
| --- | --- |
| Event Registry | Реестр событий - канонический журнал метаданных. |
| Interaction Ledger | Блокчейн взаимодействий - синоним Event Registry. |
| Lumen | Листинг - предложение товара или услуги. |
| Pact | Заказ и контракт намерения - ограничивает коммуникацию. |
| Aegis | Эскроу для сделки. |
| Trace | Акт исполнения или подтверждение. |
| Glyph | Отзыв, привязанный к сделке. |
| Rift | Спор по сделке. |
| Seal | Вердикт арбитража. |
| Brand | Санкции - слэшинг, лимиты, блокировки. |
| Lattice | Апелляция - процедурная справедливость. |
| Worm | Слой связи между узлами - устойчивость к блокировкам. |
| ANS | Amalgam Name System - имена узлов в Worm. |
