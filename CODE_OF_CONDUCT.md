# Code of Conduct

Мы строим систему - где доверие и прозрачность важнее скорости.

## Ожидаемое поведение
- Уважение к участникам - даже при конфликте интересов.
- Четкие аргументы - без личных атак.
- Документирование решений и причин.

## Недопустимое поведение
- Преследование, дискриминация, угрозы.
- Публикация приватных данных.
- Саботаж процесса рассмотрения.

## Как сообщать
Напишите в `conduct@canfly.org` (placeholder) - укажите контекст и ссылки. Ответ - до 7 дней.
