# Contributing

Этот репозиторий содержит канонические документы - изменения должны быть аккуратными и прозрачными.

## Процесс
1) Создайте issue с типом запроса - документ, спецификация, исследование.
2) Откройте PR с ссылкой на issue.
3) Обновите `CHANGELOG.md` и, если нужно - `docs/Decision_Log.md`.

## Стиль
- Короткие абзацы - строгие термины.
- Русский язык - без длинных тире.
- Используйте таблицы для схем данных.

## Рецензирование
- Минимум один reviewer.
- Для изменений политики или санкций требуется доменный Guardian обзор.
